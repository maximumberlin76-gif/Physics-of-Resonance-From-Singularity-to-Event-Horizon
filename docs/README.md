## Documentation Boundary

The documentation is organized into distinct evidence layers:

**conceptual framework → mathematical formulation → executable implementation → numerical qualification → experimental and observational testing**

These layers are connected but are not interchangeable.

A mathematical formulation is not automatically an experimental result.

A passing software workflow is not automatically physical confirmation.

A resonance interpretation is retained only where its variables, observables, controls, and falsification conditions are explicitly defined.
